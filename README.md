## 🚀 Quick Start

### Step 1: Open the App
Simply open `index.html` in your web browser to see the landing page.

### Step 2: Get API Key
1. Visit [OpenWeatherMap](https://openweathermap.org/api)
2. Sign up (it's FREE!)
3. Copy your API Key from the dashboard

### Step 3: Use the App
1. Click "Start Now" or go to `weather.html`
2. Paste your API key
3. Enter any city name
4. See real-time weather data instantly!

## 💻 How to Use

### Getting Weather Data

1. Go to `weather.html`
2. Get your free API key from [OpenWeatherMap](https://openweathermap.org/api)
3. Enter API Key and City Name
4. Click Search (or press Enter)
5. View real-time weather information

### Search Functionality

- Type any city name (e.g., London, New York, Tokyo)
- Auto-formatted with country code
- Instant results display
- Error messages for invalid cities

### Temperature Conversion

- Click °C to see Celsius (default)
- Click °F to see Fahrenheit
- Auto-converts all temperature values
- Preference is stored during session

## 📊 Weather Data Displayed

- **Current Temperature** - Large prominent display
- **Condition** - Weather description (Sunny, Cloudy, Rainy, etc.)
- **Feels Like** - Temperature that accounts for wind chill
- **Humidity** - Percentage of moisture in air
- **Wind Speed** - In km/h or m/s
- **Visibility** - How far you can see (in km)
- **Pressure** - Atmospheric pressure in hPa
- **City & Country** - Location information
- **Last Updated** - When data was fetched
