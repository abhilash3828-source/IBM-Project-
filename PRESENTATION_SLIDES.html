📋 Project Overview

WeatherVibe is a modern, responsive weather application built with:

  • ✅ HTML5 Structure
  • ✅ CSS3 Styling & Animations
  • ✅ JavaScript Functionality
  • ✅ OpenWeatherMap API Integration
  • ✅ Modern Design Trends (Glassmorphism)

🎯 Project Goals

Create a multi-page weather application
Implement real-time weather data using APIs
Design a modern, visually appealing interface
Make it responsive and mobile-friendly
Ensure smooth user experience with animations

🎨 Key Feature 1: Gradient Background

body {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  min-height: 100vh;
}

Why it matters:

  • Creates modern, eye-catching visual appeal
  • Smooth color transition from purple to indigo
  • Makes the app feel premium and professional

✨ Key Feature 2: Glassmorphism

.floating-card {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(20px);
  border: 2px solid rgba(255, 255, 255, 0.2);
  border-radius: 30px;
}

Why it matters:

  • Frosted glass effect (trendy 2024-2026)
  • Used by Apple, Microsoft, Google
  • Modern and visually sophisticated

🎬 Key Feature 3: Smooth Animations

📱 Key Feature 4: Responsive Design

@media (max-width: 768px) {
  .hero { grid-template-columns: 1fr; }
  .stats { grid-template-columns: repeat(2, 1fr); }
  .cta-buttons { flex-direction: column; }

Why it matters:

  • Flexible layouts using CSS Grid & Flexbox
  • Essential for modern web applications

🌐 Key Feature 5: API Integration

async function getWeather(city, apiKey) {
  const response = await fetch(
    `https://api.openweathermap.org/data/2.5/weather?q=${city}&appid=${apiKey}`
  );
  const data = await response.json();
  displayWeather(data);
}

Why it matters:

  • Fetches real-time weather data
  • Uses async/await for smooth execution
  • Error handling ensures reliability
